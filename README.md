## Featured project — Secure WiFi CSI Healthcare Sensing

Repository: https://github.com/shahram-h-hesari/secure-wifi-csi-healthcare-sensing

Description:

This project is a PhD research prototype exploring secure WiFi Channel State Information (CSI)-based healthcare sensing using synthetic CSI-like data. It demonstrates a reproducible Python/Jupyter/Streamlit workflow for:

- synthetic CSI generation (simulation and controlled transforms)
- baseline ML classification experiments
- adversarial robustness stress testing and analysis
- clinical-safety-aware evaluation of sensing failures and risk modes

Important note: the project uses synthetic CSI-like data only. It does not claim clinical validation, use real patient data, or imply medical-device readiness.
