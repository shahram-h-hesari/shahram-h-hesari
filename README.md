## Featured project — Secure WiFi CSI Healthcare Sensing

My featured repository is:  
https://github.com/shahram-h-hesari/secure-wifi-csi-healthcare-sensing

This project is a PhD research prototype exploring secure WiFi CSI-based healthcare sensing using synthetic CSI-like data. It demonstrates a Python/Jupyter/Streamlit workflow for synthetic CSI generation, baseline ML classification, adversarial robustness stress testing, and clinical-safety-aware evaluation.

Important note: the project uses synthetic CSI-like data only. It does not claim clinical validation, real patient data, or medical-device readiness.
