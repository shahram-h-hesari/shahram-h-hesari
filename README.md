## Featured project — Secure WiFi CSI Healthcare Sensing

[Secure WiFi CSI Healthcare Sensing](https://github.com/shahram-h-hesari/secure-wifi-csi-healthcare-sensing) is a PhD research prototype exploring secure WiFi Channel State Information (CSI)-based healthcare sensing using synthetic CSI-like data.

The project demonstrates a reproducible Python/Jupyter/Streamlit workflow for:

- Synthetic CSI generation and controlled signal transformations
- Baseline machine learning classification experiments
- Adversarial robustness stress testing and analysis
- Clinical-safety-aware evaluation of sensing failures and risk modes

**Important note:** The project uses synthetic CSI-like data only. It does not claim clinical validation, use real patient data, or imply medical-device readiness.
